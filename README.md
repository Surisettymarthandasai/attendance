AITS Student Attendance Portal 📊

An interactive, lightweight, and responsive web-based dashboard designed to view detailed subject-wise academic attendance records for students at the Annamacharya Institute of Technology and Sciences (AITS).

This portal allows faculty, administrators, or students to instantly look up individual attendance sheets, showing exact period allocations, subject breakdowns, and aggregate percentages with a single click.

🚀 Features

Interactive UI: Minimalist, user-friendly interface featuring the official Annamacharya Educational Trust insignia.

Instant Filtering: Smooth, JavaScript-driven image loading to toggle between different student profiles instantly without reloading the page.

Granular Subject Breakdown: Displays precise period breakdowns and percentage matrices for core academic subjects:

BEFA (Business Economics and Financial Analysis)

DM (Discrete Mathematics)

OS (Operating Systems)

DBMS (Database Management Systems)

SE (Software Engineering)

COI (Constitution of India)

Timeframe Accuracy: Visualized metrics capturing data specifically from 01-02-2025 to 13-02-2025.

🛠️ Tech Stack

This project is built using optimized native web technologies to ensure zero-latency execution:

HTML5: Semantic markup structuring the user interface and container systems.

CSS3: Custom stylized layouts, smooth hover transitions, and balanced container grids.

JavaScript (ES6+): Dom manipulation algorithms to load and transition student records seamlessly.

💻 How to Run Locally

Since the application utilizes pure client-side execution, no heavy server setups, compilers, or database migrations are required:

Clone the repository:

git clone [https://github.com/](https://github.com/)<your-username>/<your-repo-name>.git


Navigate into the directory:

cd <your-repo-name>


Launch the application:
Simply double-click the index.html file, or open it directly in any modern web browser (Chrome, Firefox, Safari, Edge).
